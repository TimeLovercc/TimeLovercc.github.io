博客的重新启动

2021/05/18
  • 博客

1. 前言

好久没有更新博客了。时不时会查阅一下之前自己写的博客中的内容，看到博客日期停留在了2020年10月27日，还是会想：自己好像又没能坚持做好一件事。

是也不是。我其实也没有闲着，只不过很多被动的时间消耗和催命的ddl，让自己没有时间学自己喜欢的东西并记录了。大约去年10月26日开始准备托福，准备了一个月，然后紧接着完成各项文书工作和申请工作。到了今年，忙于应付各种各样的面试以及毕业设计。另外，Workflow从Typora转换到Notion也让工作的同步变得更加简单，似乎不用博客也可以了。

其实博客还是很有用的，作为一种公开的展示自己的方式，博客是我与这个世界的一个宝贵的接口。我想，博客我要坚持写下去，作为同步的良好工具，作为费曼学习法的实践，作为与世界的接口。

2. 怎么写

我的博客分为Blog，Plan，Thoughts，Projects，About me共五个原创部分。因为有了Notion我不再需要Blog帮助同步，所以我希望我写出来的Blog能更完整，更新一点，或许是多个Notion文档的抽象和升华。Plan部分下部分再说。Thoughts部分写一些小的灵感，不需要完整，不需要实践，只要新就行。Projects部分则跟自己科研息息相关，要及时把自己的主业科研上的工作放上去。About me则是类似于个人主页的位置，及时更新自己的主业变化信息。

3. 规划系统

Plan是我的规划系统的一部分，这里把我的规划系统明确一下，方便自己。我的规划系统主要利用Notion的Milestones和Diary，Trello的年计划、月计划、周计划，以及Hexo博客的Plan部分。有时还用一下软件追踪。

固定部分

  • 年计划：Trello
  • 月计划：Trello
  • 周计划：Trello
  • 日计划：Diary

动态部分

  • 多年或跨月计划：Plan
  • 完成的较大计划：Milestones

时间管理

  • 时间追踪：ATracker Pro
  • 番茄钟：Focus
  • 日历：Google Calendar
  • 生活相关：Days Matter

4. 当前的Workflow

作为一个喜欢收集软件的人，我用过了很多软件，但是使用频率最高的可以总结一下。

  • Notion: 大部分记录、收集及文档工作。
  • Trello：日程规划。
  • Hexo: 定期总结，灵感记录，科研进展。
  • Google Calendar: 提醒。
  • draw.io : 作图。
  • VSCode: 编程。
  • Snipaste ：截屏相关。
  • Overleaf：正式写作。
  • Bitwarden ：密码管理。
  • 欧路词典：词典。
  • DeepL : 翻译。
  • PDF Expert 和 Kami：Ipad和PC阅读论文。
  • Notebility：笔记。
  • Outlook: 邮箱。
  • Chrome 和 Safari：浏览器。
  • Axmath : 公式编辑。
  • Mathpix：公式OCR。
  • Google Drive: 科研，分享相关。
  • Onedrive: PC同步盘。
  • ICloud：手机同步盘。
  • ITunes： 音乐。
